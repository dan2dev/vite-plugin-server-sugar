# Changelog

All notable changes to `vite-plugin-server-sugar` are documented here.

## 0.1.2 - 2026-06-28

- Added configurable endpoint pathname bases for generated HTTP and WebSocket
  endpoints.
- Updated the basic PWA example to use the configured pathname base.
- Expanded usage and architecture documentation.

Security: no publicly known runtime vulnerabilities with a CVE or similar
identifier were fixed in this release.

## 0.1.1 - 2026-06-28

- Included third-party notices in the published package files.
- Added release automation for package versioning, packing, publishing, and git
  tagging.
- Completed the package rename to `vite-plugin-server-sugar`.

Security: no publicly known runtime vulnerabilities with a CVE or similar
identifier were fixed in this release.
